NOTE: The API will only work for number between 0 to 999,999,999,999,999,999,999,999,999.

Covert number to English words

API: http://numandword.azurewebsites.net/api/intword/number-to-word 
Example Input: { "number": "2034000000000" }
NOTE: The input need to be a string since the number can be very large
Example Output: { "result": "two trillion thirty four billion", "message": null }

Covert English Words to number

API: http://numandword.azurewebsites.net/api/intword/word-to-number 
Example Input: { "word":"twelve thousand and three hundred and twenty one" }
Example Output: { "result":"12321","message":null }
